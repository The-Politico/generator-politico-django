generator-politico-django
=========================

A `Yeoman <http://yeoman.io/>`_ generator for building a modern front-end development environment inside a Django application.

It uses `Webpack <https://webpack.github.io/>`_, `Babel <https://babeljs.io/>`_  and `SASS <http://sass-lang.com/>`_.
